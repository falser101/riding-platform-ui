### 2.2 数据表结构

#### 2.2.1 users 用户表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | string | PK, auto | 用户ID (uuid) |
| phone | string | unique, not null | 手机号 |
| password | string | not null | bcrypt加密密码 |
| nickname | string | default "骑行者" | 昵称 |
| avatar | string | - | 头像URL |

| status | number | default 1 | 1正常, 0禁用 |

#### 2.2.2 user_tokens 用户Token表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | string | PK, auto | 记录ID |
| userId | string | FK -> users.id | 用户ID |
| token | string | unique, not null | JWT token |
| device | string | - | 设备类型 |
| expireTime | string | not null | 过期时间 |
| createTime | string | not null | 创建时间 |

#### 2.2.3 verification_codes 验证码表

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | string | PK, auto | 记录ID |
| phone | string | indexed | 手机号 |
| code | string | not null | 6位验证码 |
| type | string | not null | login/register/bind |
| expireTime | string | not null | 过期时间(5分钟后) |
| used | boolean | default false | 是否已使用 |
| createTime | string | not null | 创建时间 |

## 3. API 设计

### 3.1 认证相关 API

#### 3.1.1 发送验证码

```
POST /api/auth/send-code
Content-Type: application/json
```

**Request:**
```json
{
  "phone": "13800138000",
  "type": "login"  // login | register | bind
}
```

**Response (成功):**
```json
{
  "code": 0,
  "message": "验证码发送成功",
  "data": {
    "code": "123456"  // Mock环境返回，生产环境不返回
  }
}
```

**Response (失败):**
```json
{
  "code": 1001,
  "message": "发送过于频繁，请稍后再试",
  "data": null
}
```

**业务规则:**
- 同一手机号60秒内只能发送一次
- 验证码5分钟后过期
- 验证码使用后标记为已使用

---

#### 3.1.2 用户登录

```
POST /api/auth/login
Content-Type: application/json
```

**Request (验证码登录):**
```json
{
  "phone": "13800138000",
  "code": "123456",
  "type": "h5",

}
```

**Request (密码登录):**
```json
{
  "phone": "13800138000",
  "password": "123456",
  "type": "h5",
  "loginType": "password"
}
```

**Response (成功):**
```json
{
  "code": 0,
  "message": "登录成功",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",
    "userInfo": {
      "id": "1",
      "phone": "13800138000",
      "nickname": "骑行达人",
      "avatar": "https://example.com/avatar1.jpg",
      "role": "user"
    }
  }
}
```

**Response (失败 - 验证码错误):**
```json
{
  "code": 1002,
  "message": "验证码错误",
  "data": null
}
```

**Response (失败 - 密码错误):**
```json
{
  "code": 1003,
  "message": "密码错误，剩余尝试次数: 4",
  "data": null
}
```

**业务规则:**
- 验证码登录：验证3次错误后需重新获取
- 密码登录：5次错误后锁定30分钟
- 登录成功返回JWT Token，有效期7天

## 5. 核心实现

### 5.1 Admin - 用户状态管理 (stores/modules/user.ts)

```typescript
import { defineStore } from 'pinia'
import { getToken, setToken, removeToken } from '@/utils/storage'
import { getUserInfo, updateUser, changePassword } from '@/api/modules/user'

interface UserState {
  token: string | null
  userInfo: UserInfo | null
  loading: boolean
}

interface UserInfo {
  id: string
  phone: string
  nickname: string
  avatar: string
  role: string
  status: number
  lastLoginTime?: string
  createTime?: string
}

export const useUserStore = defineStore('user', {
  state: (): UserState => ({
    token: getToken(),
    userInfo: null,
    loading: false
  }),

  getters: {
    isLoggedIn: (state) => !!state.token,
    isAdmin: (state) => state.userInfo?.role === 'admin'
  },

  actions: {
    // 获取用户信息
    async fetchUserInfo() {
      if (!this.token) return
      this.loading = true
      try {
        const res = await getUserInfo()
        this.userInfo = res.data
      } finally {
        this.loading = false
      }
    },

    // 更新用户信息
    async updateProfile(data: { nickname?: string; avatar?: string }) {
      const res = await updateUser(data)
      if (this.userInfo) {
        this.userInfo = { ...this.userInfo, ...res.data }
      }
      return res
    },

    // 修改密码
    async changePwd(oldPassword: string, newPassword: string) {
      return await changePassword({ oldPassword, newPassword })
    },

    // 清除状态
    clearState() {
      this.token = null
      this.userInfo = null
      removeToken()
    }
  }
})
```

### 5.2 H5 - 登录页核心逻辑 (pages/login/index.vue)

```vue
<template>
  <div class="login-container">
    <van-form @submit="handleLogin">
      <van-field
        v-model="form.phone"
        type="tel"
        label="手机号"
        placeholder="请输入手机号"
        :rules="[{ pattern: /^1[3-9]\d{9}$/, message: '手机号格式不正确' }]"
      />
      
      <van-field
        v-if="loginType === 'code'"
        v-model="form.code"
        type="digit"
        label="验证码"
        placeholder="请输入验证码"
        :rules="[{ required: true, message: '请输入验证码' }]"
      >
        <template #button>
          <van-button
            size="small"
            type="primary"
            :disabled="countdown > 0"
            @click="handleSendCode"
          >
            {{ countdown > 0 ? `${countdown}s` : '获取验证码' }}
          </van-button>
        </template>
      </van-field>

      <van-field
        v-if="loginType === 'password'"
        v-model="form.password"
        type="password"
        label="密码"
        placeholder="请输入密码"
        :rules="[{ required: true, message: '请输入密码' }]"
      />

      <div class="login-type-toggle">
        <span @click="toggleLoginType">
          {{ loginType === 'code' ? '密码登录' : '验证码登录' }}
        </span>
      </div>

      <van-button type="primary" block submit-on-enter native-type="submit">
        登录
      </van-button>
    </van-form>
  </div>
</template>

<script setup lang="ts">
import { ref, reactive } from 'vue'
import { useRouter } from 'vue-router'
import { showToast } from 'vant'
import { sendCode, login } from '@/api/modules/auth'
import { useUserStore } from '@/stores/user'

const router = useRouter()
const userStore = useUserStore()

const loginType = ref<'code' | 'password'>('code')
const countdown = ref(0)
const form = reactive({
  phone: '',
  code: '',
  password: ''
})

// 发送验证码
const handleSendCode = async () => {
  if (!form.phone || !/^1[3-9]\d{9}$/.test(form.phone)) {
    showToast('请输入正确的手机号')
    return
  }
  
  try {
    await sendCode({ phone: form.phone, type: 'login' })
    showToast('验证码已发送')
    countdown.value = 60
    const timer = setInterval(() => {
      countdown.value--
      if (countdown.value <= 0) clearInterval(timer)
    }, 1000)
  } catch (error: any) {
    showToast(error.message || '发送失败')
  }
}

// 切换登录方式
const toggleLoginType = () => {
  loginType.value = loginType.value === 'code' ? 'password' : 'code'
  form.code = ''
  form.password = ''
}

// 登录
const handleLogin = async () => {
  try {
    const loginData = loginType.value === 'code'
      ? { phone: form.phone, code: form.code, loginType: 'code' }
      : { phone: form.phone, password: form.password, loginType: 'password' }
    
    await userStore.login(loginData)
    showToast('登录成功')
    
    // 跳转来源页或首页
    const redirect = router.currentRoute.value.query.redirect as string
    router.replace(redirect || '/pages/profile/index')
  } catch (error: any) {
    showToast(error.message || '登录失败')
  }
}
</script>
```

### 5.3 H5 - 个人中心页 (pages/profile/index.vue)

```vue
<template>
  <div class="profile-page">
    <!-- 已登录状态 -->
    <div v-if="userStore.isLoggedIn" class="profile-content">
      <div class="profile-header">
        <img 
          :src="userStore.userInfo?.avatar || defaultAvatar" 
          class="avatar"
          @click="handleAvatarClick"
        />
        <div class="user-info">
          <div class="nickname">{{ userStore.userInfo?.nickname }}</div>
          <div class="phone">{{ formatPhone(userStore.userInfo?.phone) }}</div>
        </div>
      </div>

      <van-cell-group class="menu-group">
        <van-cell title="编辑资料" is-link to="/pages/settings/edit-profile" />
        <van-cell title="修改密码" is-link to="/pages/settings/change-password" />
        <van-cell title="我的活动" is-link to="/pages/activity/my" />
      </van-cell-group>

      <van-button type="danger" block class="logout-btn" @click="handleLogout">
        退出登录
      </van-button>
    </div>

    <!-- 未登录状态 -->
    <div v-else class="login-prompt">
      <img :src="defaultAvatar" class="default-avatar" />
      <p>登录后查看更多信息</p>
      <van-button type="primary" @click="router.push('/pages/login/index')">
        立即登录
      </van-button>
    </div>
  </div>
</template>

<script setup lang="ts">
import { useRouter } from 'vue-router'
import { showConfirmDialog } from 'vant'
import { useUserStore } from '@/stores/user'

const router = useRouter()
const userStore = useUserStore()
const defaultAvatar = 'https://cdn.example.com/default-avatar.png'

const formatPhone = (phone: string | undefined) => {
  if (!phone) return ''
  return phone.replace(/(\d{3})\d{4}(\d{4})/, '$1****$2')
}

const handleAvatarClick = () => {
  if (userStore.isLoggedIn) {
    router.push('/pages/settings/edit-profile')
  }
}

const handleLogout = async () => {
  try {
    await showConfirmDialog({
      title: '提示',
      message: '确定要退出登录吗？'
    })
    userStore.logout()
    router.switchTab('/pages/index/index')
  } catch {
    // 用户取消
  }
}
</script>
```

## 6. 错误码定义

| 错误码 | 说明 |
|--------|------|
| 0 | 成功 |
| 1001 | 验证码发送失败（频率限制） |
| 1002 | 验证码错误 |
| 1003 | 密码错误 |
| 1004 | 手机号已注册 |
| 1005 | 验证码已过期 |
| 1006 | 登录已锁定 |
| 1007 | Token 无效或已过期 |
| 2001 | 用户不存在 |
| 2002 | 用户已被禁用 |
| 2003 | 密码强度不足 |
| 2004 | 原密码错误 |
