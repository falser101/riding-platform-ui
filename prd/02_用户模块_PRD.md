# PRD - 用户模块

## 1. 需求概述

| 字段 | 内容 |
|------|------|
| 需求名称 | 用户模块 |
| 需求编号 | REQ-002 |
| 需求类型 | 核心业务功能 |
| 优先级 | 高 |
| 状态 | 已分析 |
| 依赖 | REQ-001 (项目初始化) |

### 1.1 需求背景

骑行活动平台需要完整的用户管理体系，包括用户注册、登录、个人信息管理等功能。用户是平台的核心参与者，所有骑行活动都需要基于用户体系进行。

### 1.2 目标与范围

**目标**：建立完整的用户认证和个人信息管理功能

**范围**：
- 用户注册（手机号+验证码）
- 用户登录（手机号+验证码 / 密码登录）
- 退出登录
- 个人信息查看与编辑
- 头像上传
- 密码修改

---

## 2. 功能详细说明

### 2.1 用户注册

#### 2.1.1 功能流程

```
用户输入手机号 → 获取验证码 → 填写密码 → 提交注册 → 注册成功跳转登录
```

#### 2.1.2 功能点

| 功能点 | 描述 | 验收标准 |
|--------|------|----------|
| 手机号验证 | 输入 11 位手机号，格式校验 | 错误格式提示"手机号格式不正确" |
| 验证码获取 | 点击获取验证码，按钮倒计时60秒 | 同一手机号60秒内不可重复获取 |
| 密码设置 | 密码6-20位，需包含字母和数字 | 不满足时提示密码强度要求 |
| 注册提交 | 提交手机号+验证码+密码 | 成功返回用户ID，失败提示原因 |
| 重复注册检测 | 手机号已注册时提示 | 提示"该手机号已注册" |

#### 2.1.3 异常处理

| 场景 | 处理方式 |
|------|----------|
| 手机号已注册 | 提示"该手机号已注册"，跳转登录页 |
| 验证码错误 | 提示"验证码错误"，可重新获取 |
| 验证码过期 | 提示"验证码已过期"，重新获取 |
| 网络请求失败 | 提示"网络异常，请重试" |

---

### 2.2 用户登录

#### 2.2.1 功能流程

```
选择登录方式（验证码/密码）→ 输入手机号+凭证 → 提交登录 → 跳转首页/个人中心
```

#### 2.2.2 功能点

| 功能点 | 描述 | 验收标准 |
|--------|------|----------|
| 验证码登录 | 手机号+短信验证码登录 | 验证码正确即可登录 |
| 密码登录 | 手机号+密码登录 | 密码错误累计5次锁定30分钟 |
| 自动登录 | Token 有效期内自动登录 | 无需重复输入账号密码 |
| 记住手机号 | 记住登录手机号 | 下次打开登录页自动填充 |

#### 2.2.3 安全控制

| 规则 | 限制 |
|------|------|
| 登录失败锁定 | 同一手机号密码错误5次，锁定30分钟 |
| 验证码有效期 | 验证码有效期5分钟 |
| Token 有效期 | 7天（可配置） |
| 并发登录 | 支持多设备同时在线 |

---

### 2.3 退出登录

| 功能点 | 描述 | 验收标准 |
|--------|------|----------|
| 退出确认 | 弹出确认对话框 | 用户确认后清除本地 Token |
| 清除状态 | 清除用户信息、Token、缓存 | 返回登录页 |
| 推送订阅取消 | 退出时取消极光推送 | 防止推送无效用户 |

---

### 2.4 个人信息管理

#### 2.4.1 功能流程

```
进入个人中心 → 查看个人信息 → 点击编辑 → 修改资料 → 保存
```

#### 2.4.2 功能点

| 功能点 | 描述 | 验收标准 |
|--------|------|----------|
| 个人信息展示 | 显示头像、昵称、手机号、角色 | 信息完整，无遗漏 |
| 昵称修改 | 输入昵称，2-20字符 | 非法长度提示 |
| 头像修改 | 上传图片，支持裁剪 | 支持 JPG/PNG，最大 2MB |
| 手机号修改 | 原手机号+验证码验证后修改 | 需先验证原手机号 |
| 密码修改 | 原密码+新密码+确认密码 | 验证原密码正确性 |
| 退出登录 | 清除登录状态 | 跳转登录页 |

#### 2.4.3 头像上传规则

| 规则 | 说明 |
|------|------|
| 支持格式 | JPG、JPEG、PNG |
| 文件大小 | 最大 2MB |
| 推荐尺寸 | 200x200 像素 |
| 裁剪比例 | 1:1 正方形 |
| 上传方式 | 先压缩再上传 |

## 3. 用户故事

### 3.1 用户注册

| 用户故事 | 场景描述 | 验收要点 |
|----------|----------|----------|
| 作为新用户 | 我想注册成为平台会员 | 输入手机号、验证码、密码即可完成注册 |
| 作为新用户 | 我想确认我的密码已设置成功 | 注册后可直接使用手机号登录 |
| 作为忘记是否注册过的用户 | 我想确认手机号是否已注册 | 输入手机号点击获取验证码时能提示 |

### 3.2 用户登录

| 用户故事 | 场景描述 | 验收要点 |
|----------|----------|----------|
| 作为已注册用户 | 我想快速登录平台 | 输入手机号和验证码即可登录 |
| 作为忘记验证码的用户 | 我想用密码登录 | 支持手机号+密码方式登录 |
| 作为老用户 | 我不想每次都输入账号密码 | 7天内打开App自动登录 |
| 作为账号安全敏感用户 | 我的密码错误多次后想解锁 | 30分钟后自动解锁或联系客服 |

### 3.3 个人信息管理

| 用户故事 | 场景描述 | 验收要点 |
|----------|----------|----------|
| 作为普通用户 | 我想修改我的昵称 | 在个人中心点击编辑可修改昵称 |
| 作为普通用户 | 我想上传我的头像 | 可拍照或从相册选择，裁剪后上传 |
| 作为普通用户 | 我想修改登录密码 | 需要验证原密码后设置新密码 |
| 作为普通用户 | 我想退出当前账号 | 点击退出登录，确认后返回登录页 |

## 5. 非功能需求

### 5.1 性能要求

| 指标 | 要求 |
|------|------|
| 登录响应时间 | < 2秒 |
| 验证码发送 | < 3秒 |
| 头像上传 | < 5秒（2MB图片） |

### 5.2 安全要求

| 要求 | 说明 |
|------|------|
| 密码加密 | 使用 bcrypt 加密存储 |
| 敏感信息 | 手机号脱敏显示（138****8000） |
| HTTPS | 全站 HTTPS |
| Token 安全 | 禁止 URL 传递 Token |

## 6. 数据字典

### 6.1 用户表 (users)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | bigint | PK, auto_increment | 用户ID |
| phone | varchar(11) | unique, not null | 手机号 |
| password | varchar(255) | not null | 加密后密码 |
| nickname | varchar(20) | - | 昵称，默认为"骑行者" |
| avatar | varchar(500) | - | 头像URL |
| role | enum('user', 'admin') | default 'user' | 角色 |
| status | tinyint | default 1 | 状态：1正常，0禁用 |
| lastLoginTime | datetime | - | 最后登录时间 |
| createTime | datetime | not null | 创建时间 |
| updateTime | datetime | not null | 更新时间 |

### 6.2 用户Token表 (user_tokens)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | bigint | PK, auto_increment | 记录ID |
| userId | bigint | FK -> users.id | 用户ID |
| token | varchar(255) | unique, not null | 访问令牌 |
| device | varchar(50) | - | 设备类型 |
| expireTime | datetime | not null | 过期时间 |
| createTime | datetime | not null | 创建时间 |

### 6.3 验证码表 (verification_codes)

| 字段 | 类型 | 约束 | 说明 |
|------|------|------|------|
| id | bigint | PK, auto_increment | 记录ID |
| phone | varchar(11) | not null, indexed | 手机号 |
| code | varchar(6) | not null | 验证码 |
| type | enum('login', 'register', 'bind') | not null | 验证码类型 |
| expireTime | datetime | not null | 过期时间 |
| used | tinyint | default 0 | 是否已使用 |
| createTime | datetime | not null | 创建时间 |

## 8. 验收标准

### 8.1 注册功能

- [ ] 新用户可使用手机号+验证码+密码完成注册
- [ ] 已注册手机号再次注册时提示错误
- [ ] 验证码有效期为5分钟
- [ ] 注册成功后自动跳转登录页

### 8.2 登录功能

- [ ] 可使用验证码登录
- [ ] 可使用密码登录
- [ ] 登录失败5次后锁定30分钟
- [ ] 登录成功后跳转目标页面
- [ ] Token 有效期内自动登录

### 8.3 个人信息

- [ ] 可查看个人基本信息
- [ ] 可修改昵称
- [ ] 可上传/更换头像
- [ ] 可修改登录密码
- [ ] 可退出登录
